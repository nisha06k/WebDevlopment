OPERATOR OVERLOADING

Overloading means assigning different meanings to an operation, depending on the context. C++ allows overloading of operators, thus allowing us to assign multiple meaning to an operator. Overloaded operators are functions with special names the keyword operator followed by the symbol for the operator being defined. Like any other function, an overloaded operator has a return type and a parameter list.

Defining:

Return type classname:: Operator OperatorSymbol(arglist)
{
Body;
}

Implementation:

Operator function must be either member functions or friend functions.

Restriction:

  • Precedence and associativity of an operator should not change.
  • New operators cannot be created.
  • We cannot change the basic meaning of an operator. '+' cannot be used for subtraction.
  • Overloaded operators follow the rules of original operators.
