Strings in "C++"

C++ does not have string data type rather it implements string as a single-dimensional character array. A string is defined as a character array that is terminated by a null character'\0'. For this reason the character arrays are declared one character longer than the largest string they can hold.
Actually, you do not place the null character at the end of a string constant. The C++ compiler automatically places the '\0' at the end of the string when it initializes the array.
The standard C++ library provides a string class type that supports all the operations mentioned above, additionally much more functionality.
C++ supports a wide range of functions that manipulate null-terminated strings:

S.No Function Description
1 strcpy(s1, s2); Copies string s2 into string s1.
2 strcat(s1, s2); Concatenates string s2 onto the end of string s1.
3 strlen(s1); Returns the length of string s1.
4 strcmp(s1, s2); Returns 0 if s1 and s2 are the same; less than 0 if s1s2.
5 strchr(s1, ch); Returns a pointer to the first occurrence of character ch in string s1.
6 strstr(s1, s2); Returns a pointer to the first occurrence of string s2 in string s1.
