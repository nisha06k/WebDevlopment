DATA TYPES AND VARIABLES

Data Types

In C++ programming language there are many predefined set of data types. They are used for declaring variables or functions of different types and they are defined as the data storage format that a variable can store a data to perform a specific operation.

So mainly they are divided into two types they are
1.Fundamental data type or built in type
2.Derived data type

Fundamental data type or built in type

1 char 1 char a='A'; -127 to 127

Variables

Variables represent named storage locations, whose values can be manipulated during program run. Variable is the name of memory allocated by the compiler depending on the data type.
Declaration of variables The declaration of variable takes place in the following manner: type name. Where type is any C++ data type and name is the name of the variable. A variable must be declared before its use.
For Example:
int I; char a; float p. Here int is a data type and I is the name of the variable.
Initialization of Variables Initialization means assigning value to the already declared variables. It can be done either with declaration or after declaration.
For Example:
float a=2.456;//at declaration Float a; a=2.456;// after declaration

Scope Of Variables

A scope is a region of the program and broadly speaking there are three places, where variables can be declared:

  • Inside a function or a block which is called local variables
  • In the definition of function parameters which is called formal parameters.
  • Outside of all functions which is called global variables.
