DATA TYPES AND VARIABLES

Data Types

In C programming language there are many predefined set of data types. They are used for declaring variables or functions of different types and they are defined as the data storage format that a variable can store a data to perform a specific operation.

S.No Data Type type Explanation
1 Basics types int, char, double and float They are arithmetic types.
2 Enumerated types enum Enumeration data type consists of named integer constants as a list. It will be discussed later.
3 Void type void Void is an empty data type that has no value.
4 Derived type pointer, array, structure, union Array, pointer, structure and union are called derived data type in C language.

Enumeration

An enum(Enumeration) is a user-defined type consisting of a set of named constants called enumerators.It start with 0 (zero) by default and value is incremented by 1 for the sequential identifiers in the list.

Enum syntax in C is shown below with an sample example.

enum identifier [optional{ enumerator-list }];

Variables

C variable is a named location in a memory where a program can manipulate the data. This location is used to hold the value of the variable.The value of the C variable may get change in the program.Each variable in C has a specific type, which determines the size and layout of the variable's memory, the range of values that can be stored within that memory and the set of operations that can be applied to the variable.

Below table show some types and there sizes with ranges.

S.No Data Type Size (bytes) Range Explanation
1 char 1 -127 to 127 Typically a single octet(one byte). This is an integer type.
2 int 2 -32,767 to 32,767 The most natural size of integer for the machine.
3 float 4 1E-37 to 1E+37 with six digits of precision. A single-precision floating point value.
4 double 8 1E-37 to 1E+37 with ten digits of precision A double-precision floating point value.

A variable definition specifies a data type and contains a list of one or more variables of that type and its definition means to tell the compiler where and how much to create the storage for the variable. Variables can be initialized (assigned an initial value) in their declaration. The initializer consists of an equal sign followed by a constant expression.
