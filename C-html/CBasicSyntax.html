Input Output Functions :

C programming language provides many of the built-in function to write data. Commonly used I/O functions in C are "printf()" and "scanf()" functions which are present in stdio.h header file. We have to include 'stdio.h' file as shown in below C program to make use of these printf() and scanf() library functions. Lets study both of them in detail.

printf() & scanf() :

printf() function is used to print all data types values (Will see in coming slides.) onto the output screen. It returns the number of character printed by it. scanf() function is used to read character, string, numeric data from keyboard and it returns the number of character read by it.

NOTE:

C language is case sensitive. For example, printf() and scanf() are different from Printf() and Scanf(). All characters in printf() and scanf() functions must be in lower case.

Comments in 'C':

In the C Programming Language, you can place comments in your code that are not executed as part of the program. A comment starts with /* symbol and ends with */ and can be anywhere in your program. Comments can span several lines within your C program.

S.No Name Syntax
1 Single Line Comment 
// comment goes here 
	(or)
 /* Am E-Learning */
2 Multiple Line Comment 
/*
 * Author: Am E-Learning
 * Purpose: Multiple Line Comment.
 * Language:  C
 */
3 Comment at the end of line 
clrscr();// It is used to clear screen. 
		(or) 
clrscr();/* Am E-Learning */

Rules Of Syntax In 'C'

  • C language is case sensitive.
  • Every statement ends with semicolon (;).
  • Variables should be defined after main function opening bracket.
  • The statements in program must be written in sequencial order to get desired output.Unless any logic problem can be arise.
  • In separate statement each instruction should be written. Therefore , complete C program consist of set of instructions.
  • Whitespace is required between identifiers and keywords.
