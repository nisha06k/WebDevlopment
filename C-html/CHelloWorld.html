S.No Commands Explanation
1 #include It is also kmown as pre-processor. Its main work is to link the program with header file.
2 <stdio.h>
<conio.h>		 This are the header files used for pre-processor."stdio.h" is used for Standard Input Output and "conio.h" is used for console version which encapdulates the common I/O (Input/Output) functions.
3 void main() This is the main function from where execution of any C program begins with "void" as its return type.
4 { This indicates the beginning of the main function.
5 clrscr(); clrscr function clears the screen amd move the cursor to upper left hand corner of screen. This is located in conio.h header file.
6 /* some Comments */ whatever is given inside the command "/* */" in any C program, won't be considered for compilation and execution.
7 printf("Hello World"); printf command prints the output onto the screen.
8 getch(); This command waits for any character input from keyboard.
9 } This indicates the end of the main function.
